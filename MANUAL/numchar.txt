numchar

【name】

numchar : Read a string from standard input and convert it to a numeric character reference.

[Format]

Usage : numchar
Version : Tue Feb 19 23:38:53 JST 2019
Edition: 1

【Explanation】

Converts a string input from standard input to a numeric character reference.


【Example 1】
